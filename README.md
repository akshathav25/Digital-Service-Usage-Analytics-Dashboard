# Digital Service Usage Analytics

## Overview

This project analyzes digital platform engagement using YouTube video analytics data. The goal is to understand user engagement patterns by examining metrics such as views, likes, comments, and engagement rates.

## Tools Used

* Python
* Pandas
* Matplotlib
* Power BI

## Analysis Performed

* Identification of most viewed videos
* Engagement rate analysis
* Relationship between views and engagement
* Impact of video duration on engagement

## Dashboard

A Power BI dashboard was created to visualize:

* Top viewed videos
* Views vs engagement relationship
* Video duration vs engagement
* Engagement patterns by category

## Key Insights

* Videos with higher views tend to show stronger engagement.
* Engagement varies across different content categories.
* Video duration can influence viewer interaction levels.
